Outras formatações

Provavelmente você ja viu ou usou uma desssas canetas marca texto em seu dia a dia. Elas servem para você marcar uma parte de um texto na qual quer dar uma ênfase descomunal, ja que se trata de um trecho muito valioso para você, assim como acabamos dde fazer.

Para fazer essas marcas em HTML5, usamos a tag...para delimitar o texto que queremos demarcar, como se estivéssemos usando uma caneta marcador.

Big morreu, mas o small ainda sobrevive

Tem certas coisas que simplismente não são fáceis de entender. A tag (big) que deixava o texto maior está depreciada pelo HTML5, mas tag (small) que deixava o menor segue firme e forte

Texto deletado

Antigamente existia a tag (strike), que hoje tambem está em depreciada.No lugar entrou a tag que significa o texto está ali, pode ser lido, mas deve ser desconsideradopelo leitor.

Texto inserido

O texto inserido é exatamente o oposto do texto deletado que vimos anteriormente. Neste caso, se colocarmos a tag estamos dizendo que o texto aqui devemos prestar atenção nele./p>
